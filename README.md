 Анализ отмен бронирований

 Описание
Проект посвящён анализу факторов отмен бронирований и прогнозированию их вероятности.

 Используемые технологии
- Python (pandas, sklearn)
- SQL (SQLite)
- matplotlib

 Результаты
- выявлены основные факторы отмен
- построена модель (accuracy ~70%)
- оценены потери: 11.7 млн руб.

 Выводы
- OTA канал даёт наибольший процент отмен
- гибкие тарифы увеличивают риск отмен
