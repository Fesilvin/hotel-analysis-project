# Анализ отмен бронирований в отеле

## Описание проекта

Данный проект посвящён анализу отмен бронирований в отеле и выявлению факторов, влияющих на решение клиента отменить бронь.

Цель проекта — поддержка бизнес-решений в области управления загрузкой отеля, ценообразования и политики бронирования.

В работе используются синтетические данные, структура которых повторяет реальные данные гостиничной системы. Это сделано в целях конфиденциальности.

---

## Бизнес-проблема

Высокий уровень отмен бронирований приводит к:
- снижению предсказуемости выручки
- неэффективному планированию загрузки номеров
- потерям потенциальной прибыли

---

## Цели анализа

- определить факторы, влияющие на отмену бронирований
- выявить наиболее “рискованные” сегменты клиентов
- оценить влияние цены, типа номера и канала бронирования
- найти закономерности для дальнейшего прогнозирования

---

## Описание данных

- days_before_arrival — за сколько дней до заезда сделано бронирование  
- price — стоимость бронирования  
- room_type — тип номера  
- channel — канал бронирования  
- is_refundable — возможность возврата  
- is_cancelled — факт отмены бронирования (целевая переменная)

---

## Этапы работы

1. Генерация синтетических данных  
2. Исследовательский анализ данных (EDA)  
3. Анализ факторов отмен  
4. Визуализация зависимостей  
5. Формирование бизнес-выводов  

---

## Основные результаты

- выявлены ключевые факторы, влияющие на отмены бронирований  
- определены сегменты клиентов с высокой вероятностью отмен  
- сформированы рекомендации для снижения уровня отмен  

---

## Бизнес-ценность

Результаты могут быть использованы для:
- повышения загрузки отеля
- оптимизации ценовой политики
- снижения доли отмен
- улучшения планирования доходов
